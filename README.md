# Test Algo

## Introduction

This repo is used as a solution for an algorithmic test in Javascript.

## Installation

```npm install```
```run install```

## Usage

To launch the calculation, use 
```node index.js <string-to-test> <n>```

## Tests

To launch the tests, use
 ```yarn test```
